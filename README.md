# Y's Games - E-Commerce Website

## Tech Stack
- HTML, CSS, JavaScript
- Node.js, Express.js
- MySQL (MAMP)

## How to Run

1. Install MAMP and start Apache and MySQL
2. Open phpMyAdmin and import database/ecommerce.sql
3. Copy .env.example to .env and fill in your credentials
4. Run: npm install
5. Run: node server/server.js
6. Open: http://localhost:3000
